# Сайт про мультфильмы

Простой одностраничный сайт о мультфильмах, созданный с использованием HTML, CSS и JavaScript.

## Особенности

- Адаптивный дизайн
- Плавная прокрутка
- Интерактивная галерея
- Современный UI/UX дизайн
- Анимации при прокрутке

## Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Vanilla)
- Font Awesome иконки
- Unsplash для изображений

## Как запустить

1. Склонируйте репозиторий
2. Откройте файл `index.html` в браузере
